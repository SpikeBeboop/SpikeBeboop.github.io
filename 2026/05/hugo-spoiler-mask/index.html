有些内容不合适直接展示出来，比如影视结局，游戏剧情，小说反转，答案提示等。这类内容只需要默认遮住，读者主动悬停或点击后再显示即可

效果#

这是一段普通的文字。

这里是被遮住的剧透内容

继续写普通内容

页面显示时，剧透内容默认被遮住。

讲鼠标移动到遮罩上，或者再移动端点击遮罩后，内容会显示出来。

实现思路#

Hugo 本身不自带相关语法。

但是 Hugo 支持 Shortcode，可以自己自定义一段可复用的 Markdown 语法。

实现方式很简单：

1. 新增 spoiler shortcode
2. 在 Markdown 中用 shortcode 包住想要隐藏的内容
3. 用 CSS 将内容默认遮盖
4. hover / active / focus 时显示内容

具体实现#

新增 Shortcode#

在 Hugo 项目中增加文件：

layouts/shortcodes/spoiler.html 
<div class="spoiler spoiler-block" tabindex="0">{{ .Inner | markdownify }}</div>

这个 shortcode 用于多行内容，会把被包住的内容渲染成块级 HTML：

<div class="spoiler spoiler-block" tabindex="0">内容</div>

如果需要行内遮罩，额外新增：

layouts/shortcodes/spoiler-inline.html 
<span class="spoiler-inline" tabindex="0">{{ .Inner | plainify }}</span>

修改样式#

我们可以通过CSS控制显示效果

因为我使用的是 PaperMod 主题，修改下面文件，不存在就手动创建

assets/css/extended/blank.css 
.spoiler,
.spoiler-inline {
    position: relative;
    cursor: pointer;
    border-radius: 4px;
    user-select: none;
}

.spoiler-block {
    display: block;
    padding: 0.6em 0.8em;
    overflow: hidden;
}

.md-content .spoiler-block:not(:last-child) {
    margin-bottom: var(--content-gap);
}

.md-content .spoiler-block > :first-child {
    margin-top: 0;
}

.md-content .spoiler-block > :last-child {
    margin-bottom: 0;
}

.spoiler-inline {
    display: inline-block;
    padding: 0 4px;
}

.spoiler::after,
.spoiler-inline::after {
    content: "";
    position: absolute;
    inset: 0;
    z-index: 1;
    border-radius: inherit;
    background: var(--primary);
    transition: opacity 0.2s ease;
}

.spoiler:hover::after,
.spoiler:active::after,
.spoiler:focus::after,
.spoiler-inline:hover::after,
.spoiler-inline:active::after,
.spoiler-inline:focus::after {
    opacity: 0;
}

.spoiler:hover,
.spoiler:active,
.spoiler:focus,
.spoiler-inline:hover,
.spoiler-inline:active,
.spoiler-inline:focus {
    user-select: auto;
}

这部电影最后 {{< spoiler-inline >}}主角其实没有死{{< /spoiler-inline >}}。

这部电影最后 主角其实没有死 。

Markdown 渲染#

内部还可以正常写Markdown

例如：

{{< spoiler >}}
**加粗内容**
- 列表 1
- 列表 2
{{< /spoiler >}}
