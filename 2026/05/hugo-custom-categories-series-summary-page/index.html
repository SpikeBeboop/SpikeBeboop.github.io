前言#

最近又重新拾起来了个人博客，然后老毛病又犯了，开始鼓捣美化。

由于 Papermod 的分类页面只有 标签云 类型的分类页，单纯标签页用这种词云还不错，但是分类和专栏还是这个样子就有点怪了。

后续发了 LoveIt 主题，感觉这个里面的分类页面挺符合我的审美，于是直接开始施工

分类页面#

分类自定义模板#

layouts/categories/terms.html 
{{- define "main" }}

<header class="page-header categories-summary-header">
    <h1>{{ .Title }}<span class="categories-total">{{ len .Data.Terms }}</span></h1>
    {{- with .Description }}
    <div class="post-description">{{ . }}</div>
    {{- end }}
</header>

<section class="categories-summary-list">
    {{- range .Data.Terms.ByCount}}
    {{- $termPage := .Page }}
    {{- $pages := .Pages.ByDate.Reverse }}

    <section class="category-summary-block">
        <h2 class="category-summary-title">
                <a href="{{ $termPage.RelPermalink }}" aria-label="查看 {{ $termPage.Title }} 分类">
                    <span class="category-summary-icon" aria-hidden="true">📁</span>
                    <span class="category-summary-name">{{ $termPage.Title }}</span>
                    <span class="category-summary-count">{{ len $pages }}</span>
                </a>
        </h2>

        <div class="category-summary-posts">
            {{- range first 5 $pages }}
            <article class="category-summary-post">
                <a href="{{ .RelPermalink }}">{{ .Title }}</a>
            </article>
            {{- end }}
        </div>

        {{- if gt (len $pages) 5 }}
        <a class="category-summary-more" href="{{ $termPage.RelPermalink }}">更多 &gt;&gt;</a>
        {{- end }}
    </section>

    {{- end }}
</section>

{{- end }}

专栏页面#

专栏自定义模板#

layouts/series/terms.html 
{{- define "main" }}

<header class="page-header series-summary-header">
    <h1>{{ .Title }}<span class="series-total">{{ len .Data.Terms }}</span></h1>
    {{- with .Description }}
    <div class="post-description">{{ . | markdownify }}</div>
    {{- end }}
</header>

<section class="series-card-list" aria-label="专栏列表">
    {{- range .Data.Terms.ByCount }}
    {{- $termPage := .Page }}
    {{- $count := .Count }}
    <article class="series-card">
        <a class="series-card-link" href="{{ $termPage.RelPermalink }}" aria-label="查看 {{ $termPage.Title }} 专栏，共 {{ $count }} 篇文章">
            <span class="series-card-title">{{ $termPage.Title }}</span>
            <span class="series-card-count">{{ $count }}</span>
        </a>
    </article>
    {{- end }}
</section>

{{- end }}
